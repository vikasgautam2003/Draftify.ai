## üìñ About The Project

Draftify.ai is an AI-powered software-generation platform designed to build production-ready applications with natural-language prompts.  
You describe what you want, and Draftify AI generates:

- Components  
- Pages  
- Live preview  
- Editable code  
- Full application scaffolding  

The platform uses a real-time LLM streaming pipeline and an in-browser editor to simulate the experience of pairing with a senior full-stack engineer.

![Draftify Screenshot](Draftify.ai/public/ai.png)

## üåü Key Features

- **AI-Powered Software Generation**  
  Generates React components, UI pages, APIs, Tailwind styling, and logic from text prompts.

- **Live Streaming Responses**  
  Uses OpenAI, Gemini & Groq to stream code in real-time.

- **Built-in Sandpack IDE**  
  View, edit, and run generated code instantly with no local setup.

- **Instant Preview**  
  Live rendering of components and pages using Sandpack Preview.

- **One-Click Export**  
  Export the generated project code as a ZIP.

- **Model Switching**  
  Choose between GPT, Gemini, Groq depending on speed, cost, or accuracy.

- **Modern UI / UX**  
  Built with Next.js App Router, Server Actions, Tailwind CSS, and Framer Motion.
